CoVax E-Passport

CMR Studios

CoVax E-Passport is an application that provides reliable contact tracing for Taguig City. It is equipped with QR Code Reader and QR Code Generator.

Install 3.12MB (Android 5.1+)

App Description

CoVax E-Passport is a vaccine verification mobile application for Taguig City. The app can generate and scan QR Code instant and continuously.

Users can easily register through the app, upload their vaccination card and verify them. If users are not vaccinated, the app will display a message to get vaccinated and it will display a button that will redirect to a website where to schedule a vaccination. The app will instantly generate a QR Code that can be used to scan in establishments. There are two types of QR Codes and they are color coded.

* Red-Orange QR Code - This QR Code means that the user is not yet vaccinated or has a comorbidity.
* Green QR Code - This QR Code means that the user is fully vaccinated and do not have comorbidity

Users can also track their own travel history by navigating into the "My Profile" screen of the app.

The government and the administrator of this app can accurately report the number of vaccinated people within barangays.
